# Jogo do numero misterioso

Criado por Rafael Alves Faria.

Um simples jogo onde o objetivo principal é acertar um número misterioso, a partir de tentativa e erro.

[![Meu Vídeo](https://img.youtube.com/vi/9vJ_au0CrtA/mqdefault.jpg)](https://www.youtube.com/watch?v=9vJ_au0CrtA)

## Como jogar?

Ao acessar o site o jogador receberá uma mensagem de boas vindas a partir de uma caixa de texto, ao selecionar "OK" será pedido em uma nova caixa de texto que o jogador digite um número entre 1 e 100, com o intuito de acertar o número misterioso. 

Ao digitar um número, uma nova caixa de texto aparecerá dependendo se o usuário acertou ou não.

* Caso não tenha acertado, o programa dirá que o número é maior ou menor do que o escolhido anteriormente.
* Quando o usuário acertar o número misterioso, será apresentado uma mensagem de conngratulações.


Após isso será mostrado quantas tentativas o usuário teve até acertar o número. Então será pedido o nome, idade, o dia da semana e a linguagem preferida de programação.

***(Obs: Os dados como nome e idade __não são guardados em nenhum tipo de banco de dados__, esses dados são solicitados somente para entreternimento e aprendizado)***
